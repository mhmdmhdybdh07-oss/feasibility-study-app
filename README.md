# برنامج إعداد دراسات الجدوى | Feasibility Study Builder

نظام متكامل لإعداد دراسات الجدوى الاقتصادية - يدعم الزراعة والصناعة والطاقة

## ✨ الميزات

### 📊 الدراسات (8 متكاملة)
- تأسيس المشروع
- الدراسة الاجتماعية
- الدراسة البيئية
- الدراسة القانونية
- الدراسة التسويقية
- الدراسة الفنية
- الدراسة المالية
- الدراسة الاقتصادية

### 📚 المكتبات المرجعية
- **34 محصول زراعي** يمني بمواصفات علمية + مقارنة
- **30 مصنع تحويلي** ببيانات اقتصادية
- **10 مشاريع طاقة متجددة** + 4 حاسبات (شمسية/رياح/كهروماء/تعرفة)
- **12 مشروع نموذجي** كامل قابل للتحرير
- **قاموس مصطلحات** (28 مصطلح مالي)

### 🤖 ذكاء اصطناعي
- مساعد ذكي محادثة
- تحليل AI شامل للمشروع
- ملخص تنفيذي تلقائي

- محاكاة Monte Carlo (1000+ سيناريو)
- مخطط Tornado (تحليل حساسية)
- تحليل سيناريوهات (متفائل/متعادل/متشائم)
- SWOT + Porter's Five Forces
- إدارة مخاطر + مصفوفة حرارية
- مخطط Gantt لتتبع التنفيذ
- لوحة KPIs + رادار
- حاسبة استدامة بيئية
- خريطة موقع (OpenStreetMap)

### 🎨 تخصيص

## 🚀 التشغيل المحلي

```bash
# تثبيت
bun install

# قاعدة البيانات
bun run db:push

# تشغيل
bun run dev
```

افتح `http://localhost:3000`

## 📦 بناء ملف .exe (Windows)

### عبر GitHub Actions:
1. ارفع المشروع لـ GitHub
2. اذهب لـ Actions → "Build Desktop App"
3. اضغط "Run workflow"
4. حمّل ملف .exe من Artifacts

### عبر سطر الأوامر (Windows):
```bash
bun install
bun run build
cd .next/standalone
npm install electron electron-builder
cp -r ../../electron .
cp ../../electron-builder.yml .
cp -r ../../public .
npx electron-builder --win
```
